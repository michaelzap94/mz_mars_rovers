Class: ControllerHelper

ControllerHelper()

Represents a Controller Helper that will contain the necessary methods and data for the Controller to create the Plateau, Rovers and execute instructions.

Constructor

new ControllerHelper()

Creates a ControllerHelper. It will contain a reference to the Main and only Plateau created, and, will also hold a list of Rovers that have been created.
